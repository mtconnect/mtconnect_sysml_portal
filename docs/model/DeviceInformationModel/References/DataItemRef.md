# DataItemRef

**Supertype:** `Reference`
## Version Info

{: .auto-width }
| Introduced | Deprecated | Updated |"
|---|---|---|
| 1.4 |  |   |

[`Reference`]({% link model/DeviceInformationModel/References/Reference.md %}) that is a pointer to a [`DataItem`]({% link model/DeviceInformationModel/DataItems/DataItem.md %}) associated with another entity defined for a piece of equipment.

### Description

[`DataItemRef`]({% link model/DeviceInformationModel/References/DataItemRef.md %}) allows the data associated with a [`DataItem`]({% link model/DeviceInformationModel/DataItems/DataItem.md %}) defined in another entity to be directly associated with this entity.
