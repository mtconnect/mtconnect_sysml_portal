# Spindle

**Supertype:** `Axis`
## Version Info

{: .auto-width }
| Introduced | Deprecated | Updated |"
|---|---|---|
| 1.0 | 1.1 |   |

[`Component`]({% link model/DeviceInformationModel/Components/Component.md %}) that provides an axis of rotation for the purpose of rapidly rotating a part or a tool to provide sufficient surface speed for cutting operations.

[`Spindle`]({% link model/DeviceInformationModel/Components/ComponentTypes/Spindle.md %}) was **DEPRECATED** in *MTConnect Version 1.1* and was replaced by [`RotaryMode`]({% link model/ObservationInformationModel/ObservationTypes/EventTypes/RotaryMode.md %}).
