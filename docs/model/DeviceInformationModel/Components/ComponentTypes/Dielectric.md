[`System`]({% link model/DeviceInformationModel/Components/ComponentTypes/System.md %}) that manages a chemical mixture used in a manufacturing process being performed at that piece of equipment.

### Description

For example, this could describe the dielectric system for an EDM process or the chemical bath used in a plating process.
