# ProcessPower

**Supertype:** `System`
## Version Info

{: .auto-width }
| Introduced | Deprecated | Updated |"
|---|---|---|
| 1.4 |  |   |

[`System`]({% link model/DeviceInformationModel/Components/ComponentTypes/System.md %}) composed of a power source associated with a piece of equipment that supplies energy to the manufacturing process separate from the [`Electric`]({% link model/DeviceInformationModel/Components/ComponentTypes/Electric.md %}) system.

### Description

For example, this could be the power source for an EDM machining process, an electroplating line, or a welding system.
