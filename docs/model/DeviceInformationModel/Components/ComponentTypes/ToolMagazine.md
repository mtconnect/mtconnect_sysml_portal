# ToolMagazine

**Supertype:** `ToolingDelivery`
## Version Info

{: .auto-width }
| Introduced | Deprecated | Updated |"
|---|---|---|
| 1.7 |  |   |

[`ToolingDelivery`]({% link model/DeviceInformationModel/Components/ComponentTypes/ToolingDelivery.md %}) composed of a tool storage mechanism that holds any number of tools. 

### Description

Tools are located in [`Pot`]({% link model/DeviceInformationModel/Components/ComponentTypes/Pot.md %})s. [`Pot`]({% link model/DeviceInformationModel/Components/ComponentTypes/Pot.md %})s are moved into position to transfer tools into or out of the [`ToolMagazine`]({% link model/DeviceInformationModel/Components/ComponentTypes/ToolMagazine.md %}) by an [`AutomaticToolChanger`]({% link model/DeviceInformationModel/Components/ComponentTypes/AutomaticToolChanger.md %}).
