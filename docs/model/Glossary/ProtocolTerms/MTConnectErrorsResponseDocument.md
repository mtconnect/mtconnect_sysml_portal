# MTConnectErrors Response Document

**Supertype:** `MTConnect Response Document`

<span class="hoverterm">response document<span class="definition" data-term="response document">electronic *document* published by an *MTConnect Agent* in response to a *probe request*, *current request*, *sample request* or *asset request*.</span></span> published by an <span class="hoverterm">MTConnect Agent<span class="definition" data-term="MTConnect Agent">*agent* for the *MTConnect Information Model*.</span></span> whenever it encounters an error while interpreting an <span class="hoverterm">MTConnect Request<span class="definition" data-term="MTConnect Request">*request* for information issued from a *client* to an *MTConnect Agent*.</span></span>.
