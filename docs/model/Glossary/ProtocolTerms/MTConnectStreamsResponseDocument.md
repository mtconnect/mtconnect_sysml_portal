# MTConnectStreams Response Document

**Supertype:** `MTConnect Response Document`

<span class="hoverterm">response document<span class="definition" data-term="response document">electronic *document* published by an *MTConnect Agent* in response to a *probe request*, *current request*, *sample request* or *asset request*.</span></span> published by an <span class="hoverterm">MTConnect Agent<span class="definition" data-term="MTConnect Agent">*agent* for the *MTConnect Information Model*.</span></span> in response to a <span class="hoverterm">current request<span class="definition" data-term="current request">*request* to an *agent* to produce an *MTConnectStreams Response Document* containing the *Observation Information Model* for a snapshot of the latest *observations* at the moment of the *request* or at a given *sequence number*.</span></span> or a <span class="hoverterm">sample request<span class="definition" data-term="sample request">*request* to an *agent* to produce an *MTConnectStreams Response Document* containing the *Observation Information Model* for a set of timestamped *observations* made by *Components*.</span></span>.
